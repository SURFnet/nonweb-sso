# Federated login to non-web applications

SURFnet created easy to use software development kits that will enable developers to implement federated login in their applications. 



## Links to repositories
* [SURFnet Android SSO Library ](https://github.com/SURFnet/nonweb-sso-android)
* [SURFnet iOS SSO Library](https://github.com/SURFnet/nonweb-sso-ios)
* [SURFnet OS X SSO Library](https://github.com/SURFnet/nonweb-sso-osx)
* [SURFnet Windows SSO Library](https://github.com/SURFnet/nonweb-sso-windows)
